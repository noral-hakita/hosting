Tier-1 Network & Hardware Overview

Our infrastructure is built on bare-metal servers utilizing **AMD EPYC Milan (7003 series)** and **Intel Xeon Scalable (Ice Lake)** processors, specifically chosen for their exceptional single-core performance‚Äîthe key metric for Minecraft's core thread. We run a lightweight, custom-optimized Linux kernel (based on **Ubuntu 22.04 LTS**) to minimize OS overhead.

  • **Storage:** RAID 10 NVMe SSD arrays for maximum read/write speeds, guaranteeing sub-millisecond I/O for quick chunk generation.
  • **RAM:** ECC DDR4/DDR5 memory modules running at 3200MHz+ to prevent bit errors and ensure fast heap allocation for Java.
  • **Network:** 10Gbps uplinks at every host machine, connected to Tier-1 Asian network providers for optimal peering and shortest possible routes to Pakistan, India, and SEA regions.

Advanced Multi-Layer DDoS Protection

We employ a hybrid mitigation system combining hardware-based filtering at the network edge with custom software rulesets at the application layer. This ensures that attacks are scrubbed before they even reach your server instance.

Layer Protection Mechanism Benefit
Layer 3/4 (Network) Dedicated Hardware Scrubbing Centers, BGP FlowSpec Mitigates Volumetric Attacks (UDP Floods, SYN Floods) up to **1 Tbps**.
Layer 7 (Application) Custom Pterodactyl Firewall, Advanced Minecraft Protocol Filtering Protects against specific Minecraft exploits (Login Spam, Query Floods).
